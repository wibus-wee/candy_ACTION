##  Super Code UI 

- 14 built-in code box styles
- Two series
- Just introduce a CSS
- The global general

## 🎂 How To Use

Download `Super_Code_UI.css` into your server

Put into root dir, and put code into your html: 

```html
<link rel="stylesheet" href="Super_Code_UI.css">
```

Enjoy!
